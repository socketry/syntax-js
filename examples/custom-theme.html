Custom Theme Examples

The new CSS variable system makes it incredibly easy to create custom themes. Just override one variable (--syntax-base-hue) and the entire color scheme updates automatically using hue rotation!

Default Theme

The base theme with standard colors:

// JavaScript example
function greet(name) {
	const message = `Hello, ${name}!`;
	console.log(message);
	return true;
}

Ocean Theme

Cool blues and teals - just set --syntax-base-hue: 200:

Markdown with Ocean Theme

Markdown uses semantic token types that work perfectly with custom themes:

# Markdown Example

This is **bold** and this is *italic*.

Here's some `inline code` and a [link](https://example.com).

> A blockquote looks nice too!

- List item 1
- List item 2
