Welcome

This is a modern, framework-agnostic syntax highlighter built with Web Components. It provides automatic syntax highlighting using the <syntax-highlight> custom element.

JavaScript Example

Basic JavaScript highlighting:

// Modern JavaScript with ES6+ features class SyntaxHighlighter { constructor(options = {}) { this.theme = options.theme || 'base'; this.languages = new Map(); } async highlight(code, language) { const lang = await this.getLanguage(language); return lang.process(code); } getLanguage(name) { if (this.languages.has(name)) { return Promise.resolve(this.languages.get(name)); } // Dynamic import for on-demand loading return import(`./languages/${name}.js`) .then(module => { this.languages.set(name, module.default); return module.default; }); } } // Usage const highlighter = new SyntaxHighlighter({ theme: 'dark' }); const result = await highlighter.highlight(code, 'javascript');
